# Pagina-de-Joji
Este proyecto es una plataforma web estática dedicada a la exploración artística y discográfica de Joji (George Kusunoki Miller). El sitio combina una narrativa histórica con un análisis profundo del significado de sus obras más emblemáticas, utilizando una estética editorial "lo-fi" y minimalista.

Características
Arquitectura de Información: Estructura dividida en biografía, cronología de álbumes y análisis detallado de sencillos.

Diseño Editorial: Interfaz personalizada utilizando tipografías modernas (Bodoni Moda y Space Mono) y una paleta de colores coherente.

Componentes Interactivos: Menú de navegación dinámico (sticky nav), sistema de previsualización de imágenes y navegación fluida entre secciones.

Experiencia de Usuario (UX): Diseño adaptativo (responsive) y efectos visuales ligeros mediante el uso de texturas generadas con SVG.

Tecnologías utilizadas
HTML5: Marcado semántico para mejorar el SEO y la accesibilidad.

CSS3: Diseño avanzado mediante variables personalizadas, Flexbox y Grid Layout.

JavaScript (Vanilla): Lógica para la manipulación del DOM, control del scroll y gestión de eventos de usuario.

Estructura del Proyecto
index.html: Página principal con la biografía y discografía.

analisis.html: Sección dedicada al significado y trasfondo de las canciones.
